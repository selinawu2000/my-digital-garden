# 个人智库

## 文件夹结构
- `01-Concepts/` 概念与理论，跨来源汇集[[Rent-Seeking\|Rent-Seeking]]
- `03-SOPs/` 标准操作流程
- `04-Structures/` 分析框架
- `05-Evidence/` 数据与证据
- `06-Reading/` 书籍笔记
- `07-News/` 新闻时政，只记能联结已有知识的
- `08-Business/` 商业思考
- `_inbox/` 灵感捕捉，每周整理

## 命名规范
- 概念：`理论英文名_中文名`
- SOP：`SOP_任务名称`
- 框架：`框架名_适用场景`
- 书籍：`作者姓_书名关键词_YYYY`
- 新闻：`YYYY-MM-DD_关键词`
- 商业：`YYYY-MM-DD_主题_类型`
- Inbox：`YYYY-MM-DD_HH-MM_关键词`

## 每周Inbox整理（建议周日）
打开 `_inbox/` 文件夹，逐条处理：
- 值得保留 → 用对应模板升级为正式卡片，移入对应文件夹
- 不重要 → 直接删除
